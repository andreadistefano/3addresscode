# 3addresscode 

Flex/Bison files for generating three-address code for simple expressions.   
It supports:  

- assignment: :=
- arithmetic operators: +, -, *, /, unary -
- boolean operators: AND (&&), OR (||), NOT (!)
- relational operators: ==, !=, >, >=, <, <=
- type system: integer, real, boolean types with coercion
- evaluation
